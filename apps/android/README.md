# QuotaArc Android

This directory contains the Phase 2 Xiaomi 14 vertical slice:

- `data`: strict v1 decoding, validation, last-good cache, repository, and
  transport boundary;
- `widget`: responsive Glance UI, unique WorkManager scheduling, and manual
  refresh action;
- `app`: setup and source-separated detail screens.

## Security gate

The current Collector has no accepted authenticated device API. Release builds
therefore use a gate-closed transport, declare no `INTERNET` permission, and do
not guess the planned `/v1/*` wire protocol. Test code uses sanitized canonical
fixtures only. A real transport requires the accepted ADR, pairing and
revocation design, shared endpoint/error schemas, and Collector middleware.

The provisional application ID is `dev.quotaarc.android`. It is not a release
identity and must not be frozen until Xiaomi's interconnect package/signature
rules are verified.

## Toolchain

- Android Studio Quail 2 / compatible current stable
- AGP 9.3.0
- Gradle 9.5.0
- JDK 17
- compile/target SDK 36
- min SDK 34 (the documented Xiaomi 14 target)

This checkout includes the Gradle Wrapper generated by the pinned Gradle 9.5.0
distribution. The wrapper also verifies the recorded distribution SHA-256.
Android SDK packages and their licenses still need to be installed and accepted
by the developer before an Android build can run.

```bash
cd apps/android
./gradlew testDebugUnitTest lintDebug assembleDebug
```
